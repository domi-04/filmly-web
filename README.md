# Filmly Web

Filmly Web is a web application for movie recommendations and movie rating

## Development

Filmly Web frontend is developed using ReactJS and Bootstrap for styling.

Backend API is developed using NodeJS with Express. The same backend API is being used by Filmly Mobile app which is available on [Github](https://github.com/domi-04/filmly-mobile).

### Usage

After cloning this repository, you must install npm dependencies by running following command both inside frontend and backend directories:

```bash
npm install
```

To start ReactJS (frontend), you must run the following command inside frontend directory:

```bash
npm run dev
```

To start backend API, you must run the following command inside backend directory:

```bash
node index.js
```
